Rest API and FIX protocal integrator for Forex brokers


* Free software: MIT license
* Documentation: https://hulk.readthedocs.io.


Features
--------

1. Broker account
2. history price data
3. price streaming
4. instrument
5. order execution
6. Nightly unit testing
7. Fix protocol
