hulk
===============================

version number: 0.0.1
author: Lorne Luo

Overview
--------

Rest API and FIX protocal integrator for Forex brokers

Installation / Usage
--------------------

To install use pip:

    $ pip install hulk


Or clone the repo:

    $ git clone https://github.com/lorne-luo/hulk.git
    $ python setup.py install
